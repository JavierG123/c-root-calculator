# Calculadora de Raíz - C

Programa de consola escrito en C para calcular una función matemática compleja con menú interactivo. El usuario puede:

- Ingresar los valores para la raíz a calcular.
- Ver el resultado de la raíz calculada.
- Salir del programa.

El proyecto ha sido portado desde Borland C++ y es totalmente compatible con Windows 11 utilizando **Visual Studio Code** y **GCC/MinGW**.

## Requisitos

- Windows 11
- Visual Studio Code (gratuito)
- MinGW-w64 o TDM-GCC instalado
- Terminal con soporte ANSI (CMD o PowerShell)
